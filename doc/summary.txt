# ターン開始フェイズ
 + 消耗しているユニットを活動状態にする
 + ターン開始時の効果を解決する

# ドローフェイズ
 + カードを一枚引く。それをスキルテーブルか切り札に入れる。
 
# チャージフェイズ
 + この時点でターン終了を宣言すると、山札を復活できる
 
# ファーストスキルフェイズ
 + 各種スキルやアビリティが使える
 
# 移動フェイズ
 + ユニットを移動させる事が出来る
 + 移動したユニットは消耗状態になる
 
# 戦闘フェイズ
 + 隣接している（ライン１）ユニットを攻撃できる
 + 消耗していないユニットは、攻撃されたときに反撃できる
 
# セカンドスキルフェイズ
 + 各種スキルやアビリティが使える

# ターン終了フェイズ
 + ヒーロー以外のダメージを回復する
 + 勝利条件、敗北条件の判定を行う
 + スキルテーブルを補充する
